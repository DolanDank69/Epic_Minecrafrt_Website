    Welcome to the community page


    This page of the website is all about recognizing the amazing things that the vast Minecraft community has done. This will involve not just massive builds of castles or space ships but will involve complex redstone circuits, world records and anything that the Minecraft community has accomplished.

    Here we have a working emulation of an Atari 2600. This a complex redstone computer and 2000 command blocks and houses multiple games using dirt blocks a zeros and stone blocks are ones. Each block on the massive screen represents a pixel which means that rendering the game takes a lot of time. It does not respond to user inputs as there is no controller built into the game. It can run the game at 60 frames in one hour. The creator SethBling has done an amazing job creating a set of games inside of a single game.

    If you are into fun minigames then you will love Zombie Siege by FVDisco. The aim of the game is to kill as many zombies as possible before they reach the players end of the platform. The player uses bows and arrows and tnt cannons to kill zombies. The tnt cannons are an interesting redstone contraption; not because it is a cannon but how the range system works. There Three different buttons for three different ranges that the tnt can be launched. All of the three buttons are on the same redstone circuit line and the contraption is able to calculate the distance that the button was pressed at. FVDisco has made an awesome redstone contraption and fun minigame in the same map.
